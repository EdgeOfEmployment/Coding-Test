# 노란불 신호등 - 코드 풀이

## 문제 해결 방법

각 신호등은 **초록 → 노란 → 빨간** 순서로 반복된다.

이 문제는 **모든 신호등이 동시에 노란불이 되는 가장 빠른 시간**을 찾는 것이므로, 시간을 1초부터 차례대로 증가시키며 모든 신호등의 상태를 확인하는 방식으로 해결하였다.

---

## 1. 시간을 1초씩 증가시키며 확인

```java
for(int t = 1; t < 3200000; t++)
```

현재 시간을 `t`라고 하고, 1초부터 차례대로 검사한다.

각 시간마다 모든 신호등이 노란불인지 확인하며, 모두 노란불이라면 그 시간을 바로 반환한다.

---

## 2. 현재 신호등의 상태 계산

신호등은 일정한 주기로 반복되므로 먼저 한 주기의 길이를 구한다.

```java
int c = g + y + r;
```

현재 시간이 한 주기에서 몇 번째 초인지 계산한다.

```java
int rr = t % c;

if (rr == 0) rr = c;
```

예를 들어

- 초록 : 3초
- 노란 : 2초
- 빨간 : 4초

라면 한 주기는 9초가 되고,

시간이 11초라면

```
11 % 9 = 2
```

즉, 현재는 두 번째 초에 해당하는 상태이다.

---

## 3. 노란불인지 확인

노란불은 초록불이 끝난 직후부터 시작한다.

따라서 현재 시간이

```java
g < rr <= g + y
```

범위 안에 있으면 노란불이다.

코드에서는

```java
if (!(rr > g && rr <= g + y))
```

를 이용해 노란불이 아닌 경우를 확인한다.

하나라도 노란불이 아니면 현재 시간은 실패이므로 다음 시간으로 넘어간다.

---

## 4. 모든 신호등이 노란불이면 종료

모든 신호등이 조건을 만족하면

```java
return t;
```

를 실행하여 가장 빠른 시간을 반환한다.

끝까지 찾지 못하면

```java
return -1;
```

을 반환한다.

---

## 사용한 자료구조

- **int[][] signals**
    - 각 신호등의 초록, 노란, 빨간 지속 시간을 저장한다.

추가적인 자료구조는 사용하지 않고 반복문만으로 해결하였다.

---

## 시간복잡도

신호등의 개수를 **N**, 검사하는 시간을 **T**라고 하면

- 각 시간마다 모든 신호등을 확인하므로

```
O(T × N)
```

의 시간복잡도를 가진다.
